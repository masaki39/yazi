# Yazi Configuration
macOS personal setup.

## Dependencies

```bash
brew install git yazi micro clipboard jq poppler ffmpeg resvg imagemagick sevenzip fd ripgrep fzf zoxide starship
brew install --cask ghostty
```

## Clone

```bash
git clone https://github.com/masaki39/yazi.git ~/.config/yazi
ya pkg install
```

## Upgrade

```bash
brew upgrade yazi
ya pkg upgrade yazi-rs/plugins:smart-enter yazi-rs/plugins:toggle-pane stelcodes/bunny orhnk/system-clipboard
```
